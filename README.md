# AI Music Recommender 🎧

This is a machine learning based music recommendation system built using Python, NLP, and Streamlit.

Features:
- Song recommendation using NLP (TF-IDF + Cosine Similarity)
- Spotify-style UI
- Real-time album images using Last.fm API
- Trending songs section
- Playlist generator

Technologies Used:
Python, Pandas, Scikit-learn, Streamlit, NLP, Last.fm API
